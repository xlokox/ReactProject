================================================================================
                        TROUBLESHOOTING
================================================================================

If you encounter connection issues:

1. Verify backend is running:
   ```bash
   curl http://172.20.0.90:5001/api/test
   ```

2. Check if firewall is blocking connections:
   - Mac: System Preferences → Security & Privacy → Firewall

3. Ensure all devices are on the same WiFi network

4. Clear browser cache and restart Expo:
   ```bash
   # Kill Expo
   Ctrl+C

   # Clear cache
   npm start -- --clear

   # Or use tunnel mode
   npx expo start --tunnel
   ```

5. Check network connectivity:
   ```bash
   ping 172.20.0.90
   ```
